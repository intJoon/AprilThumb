## 2. 선택한 방법

1. **콘텐츠:** Markdown 프롬프트 + `---` 구분선 아래 AI 지시문 (웹 LLM 금지어: Cursor, node, pip 등 제외)
2. **배포:** Vercel 정적 사이트 — 접기 기본, 항목당 복사 버튼 1개, MD 미리보기 / 복사는 원문 MD
3. **확장:** **24** 전공 동등 카드 + 6 언어(선택 순서: **언어 → 전공**)
4. **콘텐츠 파이프:** `general`/`pharmacy` KO 프롬프트 디스크 + 22전공 `overlay` + `discipline-depth` + `locale-packs`(general/pharmacy **non-KO 프롬프트만**)
5. **가이드 SSOT:** `scripts/lib/discipline-guide.mjs` `fullGuideFor()` → `generate-all-content`가 **24×6** `guide.md` 생성
6. **프롬프트 SSOT:** pharmacy/general **ko** 디스크 · non-ko general/pharmacy는 locale-packs · overlay 22는 general/ko 베이스 + overlay JSON

## 3. 보류·기각한 대안

| 대안 | 기각 사유 |

## 4. UI·UX 결정

- 사용법: 읽기 전용 카드 (복사 없음)
- 프롬프트: 닫힌 상태에서 복사, 펼치면 렌더된 MD
- iOS: `viewport-fit=cover`, `theme-color`, `safe-area-inset`
- 디자인: 크림 `#faf7f2` + 틸, 십자 favicon → 겹친 카드 3장
- 피드백 패널 스크롤: `details`를 **처음 펼칠 때만** 페이지를 피드백 섹션으로 이동 · 목록 갱신(언어·전공 변경·prefetch·재시도) 시에는 스크롤 위치 유지 · 전송 후에만 최신 말풍선으로 스크롤
