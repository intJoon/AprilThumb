## 1. 문제 구체화

| 항목 | 내용 |
|------|------|
| 배경 | Cursor 스킬·서브에이전트(교수 채점, 출처 검증, caveman, 학습 동반자 등)를 약학과 지인에게 공유하려 함 |
| 제약 | 지인은 ChatGPT·Gemini **웹/앱**만 사용 — Cursor·로컬 파일·에이전트 설치 불가 |
| 1차 목표 | 복사·붙여넣기만으로 동작하는 프롬프트 패키지 |
| 확장 목표 | 전공·언어 확장, Vercel 단일 URL 공유, 브랜드 **AprilThumb** 통일 |

## 2. 선택한 방법

1. **콘텐츠:** Markdown 프롬프트 + `---` 구분선 아래 AI 지시문 (웹 LLM 금지어: Cursor, node, pip 등 제외)
2. **배포:** Vercel 정적 사이트 — 접기 기본, 항목당 복사 버튼 1개, MD 미리보기 / 복사는 원문 MD
3. **확장:** 16 전공 동등 카드 + 6 언어(선택 순서: **언어 → 전공**)
4. **콘텐츠 파이프:** `general`/`pharmacy` 베이스 + 14전공 `overlay` + `discipline-depth` + `locale-packs`(non-KO)
5. **SSOT:** `content/tracks/pharmacy/ko`는 디스크 원본 유지, `generate-all-content`가 KO 미수정

## 3. 보류·기각한 대안

| 대안 | 기각 사유 |
|------|-----------|
| Cursor 스킬 zip 공유 | 수신자 환경에 Cursor 없음 |
