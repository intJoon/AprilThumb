# 이렇게 쓰면 됩니다

**복사**를 누른 뒤 ChatGPT 또는 Gemini **새 대화** 맨 처음에 붙여넣으세요. 그다음 과제·글·질문을 이어서내면 됩니다.

자주 쓰는 프롬프트는 ChatGPT **지시사항**(Custom GPT)이나 Gemini **Gem**에 고정해 두어도 좋습니다.

## 법률 과제 팁

- **법원·조문** — 판례·법조문이 주장과 대응
- **논증 구조** — 사실·법률·결론 분리(IRAC 등)
- **해석·적용** — 본 사건·쟁점에의 적용 논증
- **인용** — 조문·판례 형식 일관·정확

## 어떤 걸 쓸까?

| 상황 | 프롬프트 |
|------|----------|
| 메모·에세이·의견서 제출 전 | 학술 채점 |
| 조문·판례가 주장을 지원하는지 | 출처 검증 |
| IRAC·쟁점 핵심 압축 | 간결 모드 |
| 논증·적용 초안 | 글 리뷰 |
| 법률 발표·세미나·포스터 | 발표 구조 |
| 법률 공부·시험·프로젝트 | 학습 동반자 |

**채점**과 **학습 동반자**는 서로 다른 대화에서 쓰는 것이 좋습니다.
