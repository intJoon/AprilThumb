**복사**를 누른 뒤 ChatGPT 또는 Gemini **새 대화** 맨 처음에 붙여넣으세요. 그다음 과제·글·질문을 이어서내면 됩니다.

자주 쓰는 프롬프트는 ChatGPT **지시사항**(Custom GPT)이나 Gemini **Gem**에 고정해 두어도 좋습니다.

## 의학 과제 팁

- **근거 수준** — RCT·가이드·전문가 견해 구분. 「일반적으로」만으로 끝나지 않음
- **병태생리·메커니즘** — 질환·치료 기전을 구체 단계로 설명
- **감별·한계** — 감별진단·부작용·금기·일반화 한계 명시
- **인용** — 참고문헌이 해당 문장을 실제로 지지. 형식 일관

## 어떤 걸 쓸까?

| 상황 | 프롬프트 |
|------|----------|
| 임상·케이스·SOAP 제출 전 | 학술 채점 |
| 가이드라인·RCT가 주장을 지원하는지 | 출처 검증 |
| 병태·약물·진단명 암기 요약 | 간결 모드 |
| 케이스 보고·SOAP 초안 | 글 리뷰 |
| 질환·케이스 발표 PPT·포스터 | 발표 구조 |
| 의학과 공부·실습·일상 | 학습 동반자 |

**채점**과 **학습 동반자**는 서로 다른 대화에서 쓰는 것이 좋습니다.
