# Mode d'emploi

Appuyez sur **Copier**, collez au début d'un **nouveau** chat ChatGPT ou Gemini, puis envoyez devoir, brouillon ou question.

Épinglez vos favoris dans les **instructions** ChatGPT ou un **Gem** Gemini.

## Conseils — informatique

- **Algorithmes et complexité** — Big-O, cas limites, correction — pas seulement « efficace »
- **Structures et conception** — Structures adaptées, compromis, alternatives
- **Correction et vérification** — Tests, preuves, preuves reproductibles
- **Sécurité et éthique** — Modèle de menace, vie privée, vulnérabilités (le cas échéant)

## Quel prompt ?

| Situation | Prompt |
|------|----------|
| Avant devoir algo / structures | Évaluation académique |
| CVE, articles sécu, citations | Vérification des sources |
| Révision concepts, Big-O | Mode concis |
| Brouillon projet / code | Relecture |

Utilisez **Évaluation académique** et **Compagnon d'étude** dans des chats séparés.

## Dans ChatGPT

- **Recherche web** pour CVE, OWASP et docs officielles.
- Affinez code et complexité dans **Canvas**.

## Dans Gemini

- **Google Search** pour CVE, Stack Overflow et docs.
- Collez longues revues de code dans **Google Docs**.

## Précaution

- **Pas de réponses complètes d'examen/devoir** — indices, concepts, prochaines étapes
- Pas de malware, contournement ni instructions de piratage
