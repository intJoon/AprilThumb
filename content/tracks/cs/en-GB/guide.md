# How to use this

Tap **Copy**, then paste at the start of a **new** ChatGPT or Gemini chat. After that, send your assignment, draft, or question as usual.

Pin favourites in ChatGPT **Custom instructions** or a Gemini **Gem**.

## computer science assignment tips

- **Algorithm & complexity** — Big-O, edge cases, correctness — not 「efficient」 alone
- **Data structures & design** — Fit-for-purpose structures, trade-offs, alternatives
- **Correctness & verification** — Tests, proofs, reproducible evidence
- **Security & ethics** — Threat model, privacy, vulnerabilities where relevant

## Which prompt when?

| Situation | Prompt |
|------|----------|
| Before algorithms / DS coursework | Academic review |
| CVE, security papers, citations | Source check |
| Exam concepts, Big-O cramming | Concise mode |
| Project or code design drafts | Writing review |

Use **Academic review** and **Study companion** in separate chats.

## In ChatGPT

- Use **web search** for CVE, OWASP, and official docs.
- Refine code and complexity notes in **Canvas**.

## In Gemini

- **Google Search** helps find CVEs, Stack Overflow, and docs.
- Paste long code reviews into **Google Docs**.

## Caution

- **Full exam/assignment answers** — hints, concepts, next steps only
- No malware, bypass, or hacking instructions
