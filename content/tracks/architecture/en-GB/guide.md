# How to use this

Tap **Copy**, then paste at the start of a **new** ChatGPT or Gemini chat. After that, send your assignment, draft, or question as usual.

Pin favourites in ChatGPT **Custom instructions** or a Gemini **Gem**.

## architecture assignment tips

- **Space & programme** — Scale, circulation, programme logic
- **Structure & materials** — Load path, materials, construction rationale
- **Environment & code** — Daylight, energy, regulations, accessibility where relevant
- **Citation** — Drawing captions and literature support claims

## Which prompt when?

| Situation | Prompt |
|------|----------|
| Before studio / pin-up work | Academic review |
| Codes, sun, literature, captions | Source check |
| Program & circulation cramming | Concise mode |
| Design statement & studio report drafts | Writing review |
| Design presentation decks | Presentation |
| Architecture study & studio work | Study companion |

Use **Academic review** and **Study companion** in separate chats.

## In ChatGPT

- **Web search** for building codes, daylight, and energy data.
- Refine drawing captions and scale notes in **Canvas**.

## In Gemini

- **Google Search** for materials, structure, and code links.
- Paste long pin-up narratives into **Google Docs**.

## Caution

- **No site/construction safety or code bypass**
