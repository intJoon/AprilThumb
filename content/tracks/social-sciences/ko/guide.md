**복사**를 누른 뒤 ChatGPT 또는 Gemini **새 대화** 맨 처음에 붙여넣으세요. 그다음 과제·글·질문을 이어서내면 됩니다.

자주 쓰는 프롬프트는 ChatGPT **지시사항**(Custom GPT)이나 Gemini **Gem**에 고정해 두어도 좋습니다.

## 사회과학 과제 팁

- **이론·개념** — 개념 조작화·이론 프레임 명시
- **방법·데이터** — 표본·설계·편향·윤리(해당 시)
- **해석·한계** — 인과 vs 상관·일반화·대안 설명
- **인용** — 문헌·데이터가 주장을 지지

## 어떤 걸 쓸까?

| 상황 | 프롬프트 |
|------|----------|
| 조사·현장·정책 보고 제출 전 | 학술 채점 |
| 출처·방법·표본 인용 검증 | 출처 검증 |
| 이론·개념·변수 요약 | 간결 모드 |
| 보고서·논증 초안 | 글 리뷰 |
| 사회과학 발표·세미나·포스터 | 발표 구조 |
| 사회과학 공부·시험·프로젝트 | 학습 동반자 |

**채점**과 **학습 동반자**는 서로 다른 대화에서 쓰는 것이 좋습니다.
